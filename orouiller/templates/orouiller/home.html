  • Modeling and Ray-Tracing Skeletal Implicit Surfaces on the GPU

     My Master Project dealt with skeletal implicit surfaces and how to model them so that we are able to raytrace them on the GPU. The question I was asking and trying to answer was: Can we implement a full rendering pipeline using skeletal implicit surfaces with ray tracing? How convenient is 3D modeling, can we animate the models easily? What about texturing? My demo showed that it was possible to do all of this quite conveniently but the caveat remains performance.
